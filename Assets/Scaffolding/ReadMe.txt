This package comes with a sample project to help you get going. The sample projects are kept within the Demo folder inside Scaffolding. Delete as you wish. Scaffolding
should be able to be incorporated in your project just by including the Scaffolding folder somewhere in your project.

For instructions on how to use Scaffolding, please see the how to video located here:
http://www.scaffoldingunity.com/quickguides/quickstart/

Accompanying documentation can be found on the Scaffolding site:
http://www.scaffoldingunity.com/
