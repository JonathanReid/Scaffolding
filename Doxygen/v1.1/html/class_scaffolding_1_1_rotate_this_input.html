Detailed Description

RotateThisInput attaches to a gameobject as a quick solution to get something to respond to the rotate gesture. Extends AbstractInput so falls under the same rules as other inputs in regards to views. E.g colliders get disabled during OnShowStart and OnHideStart phases.

Can be tested in the editor by holding down left CMD on mac or left CTRL on PC.
